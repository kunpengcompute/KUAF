## V25.3.0

### Change Description

25.3.0 is the Kunpeng BoostKit version. Software packages from this version are released as open source, and their component versions will not be updated (that is, **1.0.0** in **BoostKit-kuaf\_1.0.0.zip** will remain unchanged).

**New Features<a name="section11862975"></a>**

|Feature|Change Description|
|--|--|
|KUAF|Supports RSA and SM2 algorithms.|
