## V25.3.0

### 更新说明

25.3.0版本为鲲鹏BoostKit版本号。该版本之后的软件包开源发布，因此组件版本号不再进行更新（即BoostKit-kuaf\_1.0.0.zip中的1.0.0不会再升级）。

**新增特性<a name="section11862975"></a>**

|特性描述|更新说明|
|--|--|
|KUAF|新增支持RSA和SM2加解密算法。|
